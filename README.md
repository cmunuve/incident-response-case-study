# Incident response case study
This project simulates a small-scale security incident and documents the investigation, analysis, and response process from a SOC perspective.
## Scenario overview
A user reports that their system has become unusually slow and unresponsive. Suspicious activity is suspected and an investigation is initiated to determine the cause.
This case study documents the investigative process, evidence reviewed and response decisions.
